# Laboration 3 - frontend
Detta är frontend-delen av laboration 3 i kursen backend-baserad webbutveckling. I denna laboration har jag använt mig av samma gränssnitt och funktionalitet 
som i laboration 2, då den skulle vara densamma. Skillnaden som har gjort är att jag i denna laboration använder en NoSQL-databas istället för en relationsbaserad. 
För detta har jag använt mig av MongoDB och MongoDB Atlas, samt använt Mongoose för att skapa ett schema till min databas.
